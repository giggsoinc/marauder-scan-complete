Where the list lives
S3: config/HOOK_AGENTS/{token}/
  ├── authorized.csv ← live copy
  └── meta.json ← UI source

Device: ~/.patronai/
  └── authorized_domains ← fallback
What the agent does on each scan
1. Reads authorized_url (presigned GET)
2. Fetches fresh authorized.csv from S3
3. Overwrites local authorized_domains
4. Runs package / process / browser scan
5. Filters findings: unauthorised − authorised
6. PUTs results → ocsf/agent/scans/{token}/latest.json
Fallback if presigned URL expires (7 days)
Agent reads ~/.patronai/authorized_domains — the copy baked in at install time. Scan continues working, just won't pick up admin edits until a new package is generated.
